Installed Firefox 3 Beta 3 here today at the office. "Shiny new fixes" they said. So I load it up, and obviously none of my addons work again (so I have to run them through my converter), and I attempt to hit Curse. "Connecting to www.curse.com..." for a minute. Ok, wtf, is the website down? Nope, Google doesn't work either. Turns out, no websites loading in a timely fashion due to DNS lookups being extremely slow.

So I quickly Googled and found an answer (although unrelated it seemed).

Go to about:config and search for network.dns.disableIPv6. Change the value to true and voila, the internet is now fast again.
